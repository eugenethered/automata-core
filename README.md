# automata-core
Automata Core

## Packaging
`python3 -m build`

## Clean the build
`rm -fr dist automata.core.egg-info`

## Utilizing (dev) 
1. In the IDE
2. Go to the bottom
3. Choose "Python Packages"
4. Click on "Add Package"
5. Choose "From Disk"
6. Navigate project "dist" directory & choose the tar
